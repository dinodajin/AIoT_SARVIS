# 김대연-작은 스마트 텃밭

**기획 1: 반려식물 관리 솔루션 '나의 작은 스마트 텃밭'**

- **목적**: 데이터 기반의 식물 생육 환경 자동화 및 원격 관리 시스템 구축
- **대상**: 실내 식물 재배자, 장기 부재가 빈번한 사용자, 식물별 맞춤 관리가 필요한 초보자
- **Pain Point**: 식물별 최적 수분·광량 조절의 어려움, 관리 소홀로 인한 식물 고사, 부재 시 관리 공백
- **해결 방안**:
    - 토양 수분 및 조도 센서를 활용한 실시간 상태 모니터링
    - 식물 종별 생육 데이터(프리셋)에 근거한 자동 급수 및 LED 광량 조절
    - 카메라를 통한 식물 상태 시각적 관찰 및 앱 기반 원격 제어
- **구현 방법**:
    - **Hardware**: MCU(ESP32 등) 및 센서(수분, 조도, 온습도), 수중 펌프, 워터 밸브, 식물 성장용 LED, 카메라 모듈
    - **AI/Embedded**: 센서 데이터 수집 및 임계값 기반 제어 로직, 생육 단계별 이미지 타임랩스 기록
    - **Web/Backend**: 제어용 모바일 앱 또는 웹 대시보드, 데이터 저장을 위한 클라우드 서버 연동

식물에 따른 프리셋 세팅 제공, 직접 설정도 가능(이상적인 수분량, 일조량 등)
세팅에 따른 자동 광량 공급
센서를 통해 수분량 측정 및 공급

여기까지 LG 틔운 미니의 기능

추가:
카메라 설치와 통신을 통한, 반려식물 관찰 캠
원격으로 수분량 등 관찰도 가능
(장기간 집을 비울 때, 안심할 수 있음)

필요한 영양소 설정을 통해 적정 공급 주기 계산, 알리미

- 이게 가능한가?
영양의 적정 공급정도를 판단해서 부족하면 직접 공급(물에 섞든, 흙에 뿌리든)

- 시연 방법
시연장소에서 식물을 기르는 건 불가능.
기능 구현이 잘 되었는지의 시연이 될 것.

1. 현장에서 만들어진 제품에 마른 흙을 붓고 식물 모종 심기
2. 수분량이 적절하게 측정되는지 시연하고, 측정치에 따른 수분 공급/배수가(가능하면 영양 공급도) 적당한지
3. 측정치가 공급에 맞게 변화하는지
4. 측정치 및 관찰 영상이 관찰자에게 잘 전달되는지

# 김대연-독서실 통합관리시스템

**기획 2: 독서실 통합관리시스템 'SilentGuard & SeatManager'**

- **목적**: 소음 위치 추적 기술과 비식별 좌석 점유 감지를 결합한 독서실 운영 자동화
- **대상**: 무인 스터디카페 및 독서실 운영자
- **Pain Point**: CCTV 녹음 불법화로 인한 소음 민원 중재 근거 부재, 주관적 민원에 따른 관리 효율 저하, 실시간 좌석 점유 확인의 번거로움
- **해결 방안**:
    - 소리를 저장하지 않는 비식별 방식의 소음 발생 위치(좌표) 및 종류 분석
    - 소음 발생 로그와 좌석 점유 데이터를 매칭하여 객관적인 민원 대응 근거 제공
    - LiDAR 또는 초음파 센서를 활용한 실시간 좌석 점유 및 이용자 움직임 감지
- **구현 방법**:
    - **Hardware**: Jetson Orin Nano, I2S MEMS 마이크 어레이(4개 이상), RPLidar(공간 스캔용) 또는 좌석별 초음파 센서
    - **AI/Embedded**: TDOA(도달 시간 차이) 기반 음원 위치 추적 알고리즘, YAMNet 기반 오디오 분류 모델, LiDAR 포인트 클라우드 데이터를 활용한 객관체 탐지
    - **Web/Backend**: 실시간 좌석/소음 히트맵 웹 대시보드(Django/React), 시간대별 소음 및 점유 통계 분석, 자동 경고 알림 시스템 연동

목표 sector 중앙 천장에 마이크모듈 설치,

하방 음향신호 수집.

1. 각도와 천장 높이를 알면 음향 위치와의 거리를 알 수 있다.
2. 거리를 통해 음향 신호의 세기를 보정한다.
3. 음향신호를 분석하여 일상소음과의 gap, 패턴분석을 실행한다.
4. 음향 위치와 마이크 모듈의 vector계산을 통해 음향 신호의 위치를 특정한다.
5. log에 신호 발생 시간, 신호 세기, 소음 패턴을 기록한다.

• **확장성:** 만약 특정 구역의 감도가 낮다면, 동일한 오픈소스를 사용하는 노드를 추가하기만 하면 되므로 유지보수가 용이합니다.



# 김대연-BGM, 효과음의 시각화 확장 디스플레이

대상: 넷플릭스 등 영상을 자막 켜고 보면서 (의미심장한 음악) 자막에 열받아본 청각장애인

목적: BGM, 효과음 등을 시각화하여 청각장애인의 영상 시청 몰입에 도움을 줌

방법: 기존의 재생매체에 확장 디스플레이를 붙이는 식으로, 적절한 특수효과 출력

![image.png](attachment:7b15ec11-5898-466b-a854-4005387720a3:7664f636-1316-4cd4-bf0b-8c33701663e9.png)

이런 느낌으로, 타격음이나 깨지는 음 있으면 그것도 시각화로 표시

AI가 영상을 직접 분석하거나, 자막에 (깨지는 소리) 이런거 인식해서 적절한 시각화를 통해 확장 디스플레이에 표시.

고민 지점:

1. 마이크/ 카메라로 영상 정보를 수집해서 → 시각화 데이터를 생성해서 → 디스플레이에 띄운다.

장점: 

높은 호환성, 디스플레이는 기존의 것을 쓰고 처리 장치만 붙이면 됨

단점: 

마이크/카메라가 달린 정보 처리 및 생성 장치가 별개로 있어야 함

지연시간 길 우려 있음.

1. 재생 매체(TV, 태블릿 등)에 연결하는 확장 디스플레이. 디스플레이에서 영상 정보를 미리(혹은 실시간으로) 읽고 처리해서 출력까지 한다.

장점:

지연이 적음

기계가 간소함

단점:

이 기능을 위한 디스플레이를 따로 제작해야 함


최종적으로, 여러 차례 상담과 회의를 통해 팀원의 기획- 싸피스를 채택하여 확정하였다.
