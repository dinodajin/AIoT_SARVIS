# 2026.01.14. 오늘 한 일 
기획은 '싸비스'로 확정 지었고, 기능을 크게 하드웨어와 앱으로 분리하여 저는 앱 기능 파트를 맡았습니다.

앱 명세서 (미완성) 
- 앱과 싸비스 연동 : 앱 실행 시 싸비스 기기와 연결되고 연결 상태를 표시함
- 주변 환경 등록 : 카메라로 현재 주변 상황을 촬영하여 싸비스가 구동 가능한 범위를 설정함 (3D) 
- 싸비스 환경 설정 : 기본 설정 기능 (얼굴과 디스플레이와의 거리, 디스플레이 위치, 디스플레이 이동 속도 등 사용자화) 
- 권한 상태 관리 : 카메라 권한 상태 확인 / 권한 미확인 시 안내 표시 
- 로봇에서 입력 받은 신호 처리 : 휴대폰 화면 밝기, 음량 조절, 영상 되돌리기 등의 기능을 로봇에서 받은 신호에 따라 처리함 (OS 제공 API 활용) 
- DB 설계 : 디바이스 ID, 카메라 권한 등의 정보 기록된 DB 설계

+ 앱을 구현하기 위해 Kotlin 공부를 시작했습니다.
- **‘Joyce의 안드로이드 앱 프로그래밍 with 코틀린’** 책으로 공부했습니다

### Kotlin 기초

- 코틀린은 자바처럼 JVM (Java Virtual Machine) 에서 동작하는 프로그래밍 언어임
- 특징
    - 간결함: 기능이 다양한 표준 라이브러리를 제공해 상용구 코드를 적게 작성하면서도 아이디어를 풍부하게 표현함
    - 더 안전한 코드: 널 포인터 예외와 같은 일반적인 프로그래밍 실수를 방지하는 다양한 장치 제공함
    - 호환성: 자바와 100% 호환되므로 자바로 작성된 코드도 자유롭게 사용 가능함. 자바 ↔ 코틀린 호환 가능
    - 구조화된 동시 실행: 코틀린 코루틴을 사용하면 비동기 코드를 쉽게 사용할 수 있음. 코루틴을 이용하면 네트워크 호출, 로컬 데이터 액세스에 이르는 백그라운드 작업을 효과적으로 간단히 구현하고 관리 가능함

(+ 안드로이드 앱을 만드는 데에 **안드로이드 스튜디오(안드로이드 앱 개발용 IDE)** 를 사용함)
