# 2026.01.12. 기획 내용
## 구상해 본 기획 아이디어

### 비상탈출 경로 자동 생성 시스템

로봇이 건물을 매핑하면서 동시에 ‘지금 불이 났다면 어디로 대피해야 할까?’ 를 실시간으로 계산. 로봇이 구역을 돌면서 실시간으로 지도 생성 > 특정 지점에 가상 화재 발생 표시 > AI 가 즉시 모든 위치에서 출구까지의 최적 대피 경로를 계산 > 화면에 화살표로 대피 경로가 그려짐 > 앱으로 자기 위치를 선택하면 맞춤형 대피 경로 표시

1. 로봇이 건물 매핑 → SLAM
2. 가상 화재 발생 
3. 위험 영역 추론 → AI 활용 (시계열 데이터 기반 위험 확산 예측)
4. 공간 상태 추론 → AI 활용 (이 통로는 지금 안전한가?, 30 초 뒤에도 안전할까?, 통로별 위험 점수 추론)
5. 개인 맞춤 경로 의사결정 → AI 활용 (현재 이 사람에게 가장 생존 확률이 높은 선택은?, 최단 거리 X, 최소 위험 X, 조건부 최적 선택) 
6. 실시간 재계산 판단 → AI 활용 (지금 경로를 유지할지, 새로 계산하는 게 이득일지)  
7. 시각화

→ AI를 활용할 수 있지만 IoT가 활용되어야 할 이유가 좀 미약하다.

→ 기획 개선: 상황을 공연장으로 가정

- 공연 전: 이동 로봇이 SLAM 을 통해 해당 공연에 맞는 공간 지도를 생성함. 무대, 장비, 임시 구조물을 반영해 출구와 통로 그래프를 구성

- 공연 중: 군중 밀도, 이동 속도, 병목 구간을 실시간으로 감지해 혼잡 패턴을 축적하고, 압사나 병목 위험이 커지는 구역을 사전에 탐지함

- 비상상황 발생 시: 화재나 위험 지점을 입력으로 받아 위험 확산을 AI로 예측, 통로 별 안전성을 평가해 특정 출구에 사람이 몰리지 않도록 군중을 분산하는 대피 경로를 계산, 그 결과를 스크린이나 바닥 화살표를 통해 직관적으로 안내함

### 헬스케어

지역 감염병 확산 예측 시스템 - 공공장소에 체온 센서, 기침 소리 감지 센서를 설치하고, 유동인구 데이터, 기상 데이터, 과거 감염병 발생 패턴을 분석해서 특정 지역의 감염병 확산 가능성을 조기 경보

### 웨어러블 기반 지역 건강 지도
