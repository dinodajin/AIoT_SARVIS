# Onshape 3D CAD 기초 학습 및 OMX-AI(leader) STEP 파일 오류 해결

1. HW를 담당하기 위하여 3D CAD 기초 학습을 하였습니다.
2. 이용하려는 Robotics 파일 편집이 되지 않아 오류를 해결하였습니다.


### 수강한 Onshape 튜토리얼 목록

#### Beginner Tutorial Series
1. **Creating Sketches and Objects**
   - 스케치 생성
   - 기본 형상(Extrude) 생성
   - 기준면(Plane) 이해

2. **Adding Features**
   - Fillet, Chamfer 적용
   - Hole, Pattern 등 부가 기능 사용
   - Feature 기반 설계 개념 이해

3. **Other Tidbits**
   - Constraint 활용
   - Sketch 수정 및 관리
   - 설계 안정성 확보 방법 학습

4. **Views and Orientation**
   - View Cube 사용
   - 기준 좌표계 이해
   - 모델 시점 관리

5. **Documents**
   - Onshape 문서 구조
   - Part Studio / Assembly 개념
   - 협업을 고려한 문서 관리

6. **STL Export for 3D Printing**
    - STL 파일 내보내기
    - 단위(Unit) 및 해상도 설정
    - 3D 프린팅을 고려한 모델 확인

### STEP 파일 분석 내용.

기존의 파일은 stl로 구성되어 편집이 되질 않았다. 

-> stl 파일은 면이 아닌 삼각형으로 구성된 파일이라 편집이 되질 않는다.

-> 따라서 step로 된 파일을 찾아서 요소를 분석하였다.
