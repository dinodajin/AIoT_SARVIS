# SARVIS UI/UX 개발
###  설정, 제어 등 전반적인 화면 수정 및 기능 수정을 하였습니다.

---

## 주요 기능

### 1. 사용자 인증 흐름
- 얼굴 인식 기반 회원 가입 (시뮬레이션)
- 음성 등록 / 건너뛰기
- 3D 가동 범위 설정
- 게스트 모드 지원

### 2. 로봇 제어
- 얼굴 추적 시작 / 중지
- 사용자 위치로 이동
- 시야 비켜주기
- 수동 방향 제어 패드
- 각도(좌우/상하), 거리 슬라이더 제어

### 3. 음성 명령
- 음성 명령 ON/OFF 토글
- 웨이크 워드 `"싸비스"` 시뮬레이션
- 명령 예시: 따라와, 비켜줘, 정지, 여기 와
- 음성 상태 시각화 (Listening / Active)

### 4. 프리셋 시스템
- 현재 각도·거리·속도 설정 저장
- 프리셋 선택 및 즉시 적용
- 프리셋 수정 / 삭제
- 기본 프리셋 보호

### 5. 설정 화면
- 사용자 프로필 표시
- 얼굴/음성 재등록
- 추적 속도 설정 (느림 / 보통 / 빠름)
- 프리셋 개수 확인
- 로그아웃
