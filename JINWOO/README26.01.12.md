# 냉장고를 부탁해
- 카메라로 냉장고에 남은 재료를 파악 현재 가능한 레시피 추천 , 또는 약간의 재료 추가로 만들 수 있는 추천 레시피를 보여주는 프로그램

 -1. 냉장고 내부 사진, 오늘 장본 것을 간단 등록
 -2. 객체탐지로 라벨링된 품목 감지 감지 결과를 저장
 -3. 레시피 db를 재료,조리법으로 저장
 -4.현재 재료로 만들 수 있는 레시피,부족한 재료가 몇가지 있는 레시피를 추천

# 주차의 달인
- 주차를 최대한 공간 활용 할 수 있도록 주차장 상단에서 어떻게 해야 할지 핸들 각도 이동 거리등을 판단,공유 하는 프로젝트

-1. 주차장 상단 고정 카메라로 영상 입력(1대)
-2. 바닥 좌표 보정: 카메라 캘리브레이션 + Homography로 탑뷰(평면) 좌표계로 변환
-3. 차량/주차면 인식
차량: 객체탐지로 위치(중심) + 크기
차량 각도: 바운딩박스 방향/코너 추정으로 대략적인 yaw 계산
주차면: (고정 환경이면) 미리 좌표 등록 
-4. 가이드 생성
현재 상태(위치/각도) → 목표 주차면(중심/각도)로 가는 경로 생성
출력: “좌로 더”, “우로 더”, “전진 n m”, “후진 n m” 같은 단계 지시

- 피드백 : 주차는 출차도 고려해서 실용성이 떨어져 보임

# 차량 통신
- 차량간 통신 / 에어드롭 과 유사하게 차량간 통신을 통해 앞 뒤 차량간 간단한 통신 수행

근거리 발견 + 연결

Wi-Fi Direct(또는 BLE로 발견 후 Wi-Fi로 전송)로 주변 차량(폰/라즈베리파이)을 탐색
-2. 세션/보안 최소 구현
최초 1회 페어링(PIN/QR) → 이후 토큰으로 재연결 -> 보안부분과 편의성 부분에 의문
-3. 메시지 전송 계층 구현
텍스트: UDP/TCP로 즉시 전송
-4. 기능 로직(앱 레벨)
“앞차/뒷차에 보내기” UI + 프리셋(급정거, 사고등)
