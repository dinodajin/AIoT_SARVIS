# Project 1. Spatial-Responsive Auditory Interface (SRAI) [공간 반응형 청각 인터페이스 및 능동형 위험 전파 시스템]

## 프로젝트 개요
- 공장이나 건설 현장의 소음에 대한 작업자들의 불편함을 해결하고자 생각하게 되었다
- 이러한 고소음 산업 현장에서 작업자의 청각적 고립 문제를 해결하기 위해, 특정 소리만 선택적으로 필터링하는 기술과 위험 상황을 물리적으로 전파하는 RC카를 결합한 안전 통합 솔루션을 기획하였다

## 핵심 기능
Selective Audio Filtering (On-Device AI): 산업 현장의 상시 소음은 억제하고, "살려주세요", "쿵" 등 특정 위험 키워드 및 충격음만 실시간으로 추출하여 작업자에게 전달

Emergency Event Propagation (V2M): 위험 상황 감지 시, BLE/Serial 통신을 통해 현장 순찰 로봇(RC카)에 비상 명령 내림

Active Patrol & Response: 로봇이 사고 지점으로 즉시 이동하여 사이렌 및 LED로 주변에 위험 전파

Admin Remote Communication: 관리자가 원격으로 로봇을 조종하며 특정 작업자의 이어폰으로 직접 음성 메시지 송출

## 기술 스택
Hardware: MCU (STM32/ESP32), RC-Car Chassis, I2S Mic, Bone Conduction/Earphone

Communication: BLE 5.0, Serial (UART), Wi-Fi (Admin Dashboard)

Algorithm: FFT (Fast Fourier Transform), Audio Event Detection (TinyML)

# Project 2. V2V Bio-Directional Sound & Community Platform [V2V 지향성 음성 통신 및 번호판 인식 기반 차량용 메신저]

## 프로젝트 개요
- 밖으로 울리는 클락션 소리에 불편함을 느껴 "클락션 소리를 상대방 차량안에서만 들리게 할 수 는 없을까?" 라는 생각에서 시작하게 되었다
- 불필요한 소음을 줄이고, 주변 차량과의 직접 소통을 통해 도로 위 소통의 문화를 개선하는 V2V 인터랙션 시스템을 기획하였다
- 차량 번호판을 매개체로 소통 채널을 연결하여 불필요한 마찰을 줄이고 매너 있는 운전 환경을 제공할 수 있다

## 핵심 기능
Selective Horn (In-Cabin Alert): 클랙슨을 울리면 주변 차량의 스피커로 직접 경고음을 전송하고, 도로 전체의 소음은 줄이되 타겟 차량에게는 확실한 알림 전달

Real-time License Plate Mesh Networking: 카메라를 통해 주변 차량의 번호판을 식별하여 해당 차량과 즉시 1:1 또는 그룹 채팅방 개설

Social Driving Interaction: "고맙습니다", "먼저 가세요" 등 정형화된 매너 사운드를 비상 깜빡이 대신 음성/데이터로 전송

AI Traffic Heatmap: 특정 시간/구간별 클랙슨 발생 빈도를 AI로 분석하여 도로 정체 및 위험 구간 예측 데이터 제공

## 기술 스택
Hardware: Raspberry Pi / Jetson Nano (for Image Processing), GPS, RF Module

Software: OpenCV (License Plate Recognition), MQTT/P2P Protocol

Data: Time-series Data Analysis for Traffic Noise Heatmap
